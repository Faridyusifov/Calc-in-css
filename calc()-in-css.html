Calc() function in CSS

Calc() is for values

calc() funksiyasini yalniz values(deyerler) hissesinde istifade ede bilersiz. Meselen :

.class{
  font-size: calc( 12px + 4px );
  height: calc( 70% - 20px );
  width: calc( 100% - 20% );
  margin: calc( 40px + 20vw );
}

The Math operators of calc()

calc-in riyaziyyat operatorlarinda + , - , * , / var. Amma onlardan nece istifade etmeyi de bilmek lazimdi

Toplama (+) ve cixmada (-) her iki reqem uzunluq olmalidir. Meselen :

.class{
        /*True*/
  margin:calc( 20px + 2vw ) ;

        /*False*/
  margin:calc( 12px + 8 ) ;
}

Toplama ve cixmada boshluq vacibdir.

.class{
            /*True*/
      font-size:calc( 20px + 12px ) ;

            /*False*/
      font-size:calc( 20px+12px ) ;

            /*True*/
      font-size:calc( 30px - 5px ) ;

            /*False*/
      font-size:calc( 30px-5px ) ;
}

Bolmede (/) ikinci eded reqem olmalidir, uzunluq ve 0 olmaz. Meselen :

.class{
        /*True*/
  margin:calc( 60px / 5 ) ;

        /*False*/
  margin:calc( 50px / 10px ) ;

        /*False*/
  margin:calc( 50px / 0 ) ;
}

Vurmada (*) bir eded reqem olmalidir. Meselen :

.class{
        /*True*/
  margin:calc( 20px * 3 ) ;
    
        /*False*/
  margin:calc( 10px * 5px ) ;
}

Nesting calc(calc());

.class{
    height: calc( 
        calc( 100% / 10 )
        +
        calc( 2rem * 4 )
        );
}
.class{
  width:calc( 20px * 3 - 35px / 7 ) ;
}
